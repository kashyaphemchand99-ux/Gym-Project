## 🎯 API Endpoints Overview

### Authentication
- POST /api/auth/register
- POST /api/auth/login

### User
- GET /api/user/profile
- PUT /api/user/profile

### Memberships
- GET /api/memberships/plans
- POST /api/memberships/purchase

### Trainers
- GET /api/trainers
- GET /api/trainers/:id

### Classes
- GET /api/classes
- GET /api/classes/:id

### Bookings
- POST /api/bookings
- GET /api/bookings
- DELETE /api/bookings/:id

### Utilities
- POST /api/bmi/calculate
- POST /api/contact

## 🚨 Troubleshooting Quick Tips

### Server won't start
- Make sure port 3000 is available
- Try: `netstat -ano | findstr :3000` to check
- Change port in `server.js` if needed

### Database errors
- Delete `gym.db` and restart (will recreate)
- Check database.js for schema issues

### API calls failing
- Check browser console (F12)
- Verify server is running
- Check API URL in api.js

### Pages not styling
- Refresh browser (Ctrl+Shift+R)
- Check style.css exists
- Clear browser cache
